# Image Mixer & Beamforming Simulator

## 📌 Overview
Interactive system for image frequency analysis (FT) and beamforming simulation using real-time visualization.

# 🖼️ Part A – Image Mixer

<img width="1911" height="878" alt="mix with different weights" src="https://github.com/user-attachments/assets/40a7befd-26ce-4bab-9f78-42ce0bb9d185" />

## Features
- View FT components:
  - Magnitude
  - Phase
  - Real
  - Imaginary
- Weighted FT mixing + IFFT reconstruction
- Select frequency regions (low/high)
- Brightness & contrast control
- Real-time updates with threading

# 📡 Part B – Beamforming Simulator

## Features
- Linear & curved phased arrays
- Adjustable elements & delays
- Real-time beam steering
- Wave interference visualization
- Multiple array support
- Predefined scenarios (5G / Ultrasound / Focused beam)
